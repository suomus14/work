2018年CIS方針は「戦略的転換を推進し、自走できる事業会社へ成長する」と掲げている。その実現には、各職場が環境変化に応じて迅速かつ自律的に課題解決を行える組織へ成長することが不可欠である。G3等級には、進取の気性と三自の精神を体現し、組織のけん引役として成果を出すことが求められる。私はキヤノンの半導体露光装置を統合的に支援する「Lithography Plus」の開発および市場トラブルサポートを担っており、従来の縦割り的な問い合わせフローを見直し、関係者が一体となる協働体制へ転換することが、組織成長に向けて不可欠であると考えている。

Lithography Plusはキヤノンの半導体露光装置の安定稼働を支える重要なソリューションであり、市場トラブルの影響は顧客の生産計画に直結する。従来、トラブル発生時には「顧客→現地スタッフ→CS→市場トラブル対応チーム→CIS(私)」という多段階の経路を経由して対応していたため、ログ取得や暫定対応まで平均３〜４週間を要していた。原因究明や対策実施の遅れが顧客影響や信頼低下を招いていた。また、各段階で情報に齟齬が生まれ、対応の優先順位や緊急度の共有も十分でなかったため、復旧に時間を要していた。

この課題に対し、私は進取の気性をもってフロー改善に着手した。まず、現地法人、CS、市場トラブル対応チーム、CIS(私)の関係者を集めた定例会議を設置し、さらにTeamsチャネルを開設して情報共有の即時化を図った。その結果、従来はリレー形式で順に伝達されていた問い合わせが、同時に全関係者へ伝わるようになり、情報の遅延や認識合わせの手間が大幅に削減された。加えて、FTAをトラブルごとに作成し、現地スタッフが必要なログを取得し、暫定復旧操作まで自律的に行える体制を構築した。これにより、ログ取得や暫定対応のリードタイムは従来の２〜３週間から約１週間に短縮され、問い合わせの往復も削減された。

取り組みを推進する過程では、三自の精神を実践した。自覚として、対応遅延が顧客信頼を損なう重大課題であることを認識し、改善の必要性を関係者全員に共有した。自治として、定例会議やTeamsチャネルの運用ルールを整備し、関係者が自律的に行動できる仕組みを定着させた。自発として、従来のフローに依存せず、自ら新しい協働体制とFTA運用を提案・実行した。この結果、現場の即応力が向上し、対応件数や解決スピードが改善された。さらに、FTAや改善事例をナレッジとして蓄積することで、同種トラブルの対応スピードが向上し、対応精度の平準化にも寄与した。

今後は、この協働体制とFTA運用を基盤として、トラブルの早期検出とナレッジ活用の継続を進めることで、より迅速かつ安定した市場対応体制を確立する。具体的には、トラブル予兆を分析するデータ収集や監視ツールを導入し、発生前に対応を開始できる仕組みを整備する。また、過去事例や対応手順を体系化し、ナレッジとして蓄積・共有することで、チーム全体の判断力と対応力を高め、属人的な依存を排除する。これにより、Lithography Plusの安定稼働を維持しつつ、現場が主体的に判断・行動できる組織力をさらに強化できると考える。

私は、進取の気性と三自の精神を実践しながら、チームをけん引し、組織の自律的な課題解決力を高めることに貢献する。今回の取り組みと今後の展望を通じて、市場トラブル対応を単なる後処理から、組織全体の成長と顧客信頼維持に直結する重要な活動へと進化させることが可能である。

予実・課題管理について
障害対応において要件ごとに見積もりと実績工数を管理し、予実精度の向上に取り組んだ。従来は見積もり形式が曖昧だったが、テンプレート化により標準化と簡略化を実現。さらに過去の予実データを参照し、妥当性の高い工数算出へ改善した。
緊急対応における計画遵守について
委託元の緊急トラブル時にはサポートを実施。その際、別案件の進捗遅延が懸念されたため、事前に委託元へ相談し日程調整を行った。結果として全体計画への影響を回避し、計画通りの日程を遵守できた。
作業効率化の取り組み
リソースのグラフ化ツール導入やCoverityのGit連携により、業務効率化に貢献した。

CM活動や開発革新など、受託稼働に直接は係らない作業が多く発生した状況下でも、受託稼働率は75％以上を維持し、さらにCM活動を除いた場合には85％以上を達成することができた。
案件の谷間や作業待ちが生じた際には、提案型のアプローチで追加要件を受注し、稼働率向上に寄与した。
これらの取り組みにより、非受託業務との両立を図りつつ、課業目標を達成できた点は大きな成果であり、安定した稼働体制の確立に貢献できたと考える。

MDAS案件においては、評価障害は発生したものの、仕様の再検討を要する手戻りはなく、計画通りに作業を進めることができた。
要求仕様書に対する疑問点の洗い出しや機能の事前調査を十分に行い、課題やリスクを委託元と相談しながら事前に解消・低減したことが成果につながったと考える。
発生した障害は単純なミスによるものであったため、今後はチーム内ルールの明確化や新たな運用ルールの策定を進め、再発防止に取り組んでいきたい。
これらの活動を通じて、目標である「手戻りゼロ」にさらに近づけるよう努める。

下期は開発革新や安全衛生に関する活動はなかった。一方で、CM活動はクロージングに向けた作業が中心となり、今年の工数配分における課題点やCM間の負荷分散策について事務局へ提言を行った。また、来期に備えて課題点の整理や引継ぎ事項をまとめ、準備を進めることができた。来年は別人員が担当となるため、これらの取り組みを通じて円滑な引継ぎが可能となるよう意識して活動を行った。



今年度は障害対応において初めてプロジェクトリーダーを担当し、工数管理、日程調整、顧客との折衝、進捗報告など幅広い業務を一手に担った。初めての経験が多い中でも大きな問題なく完遂し、責任ある立場で計画通りに業務を進めることができた。特に委託先との折衝では要件の整理や工数見積もりの妥当性確認を行い、透明性のある進行管理を実現した。また進捗報告では課題やリスクを早期に共有し、関係者との認識合わせを円滑に進めることで、手戻りを防止し安定したプロジェクト運営につなげた。これらの経験を通じて、リーダーとしての調整力や課題対応力を実務で発揮できたことは大きな成果である。さらにCM活動では新入社員向け集合研修の指導を担当し、業務の進め方や課題対応の考え方を伝えることで新人育成に貢献した。指導を通じて自身もコーチングスキルを磨く機会となり、教育面での役割を果たすことができた。これらの取り組みにより、プロジェクトリーダーとしての経験を積み、課題対応力と新人育成力を発揮した一年となった。来年度はこの経験を基盤に、より高度なリーダーシップを発揮し、安定した業務遂行と人材育成に寄与していきたい。

今年度は業務遂行においてプロセス面での改善と実践を重視した。まず PDCA では、障害対応において要件ごとに区切られた進行に合わせ、直前の予実を参考に次要件の見積もりを行うことで精度を高めた。さらに前要件で得た知見をもとにFTAを作成し、効率的な原因特定につなげることができた。報連相 では、関係部門間での情報リレーにおける認識齟齬や伝達ミスを防ぐため、関係者全員を集めた定例会を実施し、認識合わせを行った。これにより円滑なコミュニケーションを確保し、計画通りの進行を支えた。リソース活用 の面では、グラフ化ツールやCoverityのGit連携を導入し、工数や進捗を可視化することで効率的なリソース配分を実現した。案件の谷間では追加要件を提案し、稼働率向上にも寄与した。関連部門との折衝 では、委託元と事前に日程調整を行い、全体計画を遵守する体制を整えた。部下後輩の育成 では、CM活動の一環として新入社員向け集合研修の指導を担当し、業務の進め方や課題対応の考え方を伝えることで新人育成に貢献した。指導を通じて自身もコーチングスキルを磨き、教育面での役割を果たした。

障害対応においては、関係者を集めた定例会の場で、各障害に対する調査方針の策定や現地への指示出しを私が主導した。主体的に調査の進め方をコントロールできたことで、日程調整や業務量の管理を円滑に進めることが可能となった。その結果、委託元は余力を他業務に振り向けることができ、双方にとって効率的かつ有益な対応となった。プロジェクトリーダーとして、調査方針の提示と指示出しを通じて関係者全体の動きを整理し、負担軽減と成果確保を両立できた点は今年度の大きな成果である。
